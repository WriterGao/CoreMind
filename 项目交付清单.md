## 🚀 快速开始

### 开发环境启动

```bash
# 1. 配置环境变量
cp .env.example .env
# 编辑.env文件，填入必要配置

# 2. 启动服务
make dev

# 3. 初始化数据库
make init-db

# 4. 访问应用
# 前端: http://localhost:5173
# 后端: http://localhost:8000
# API文档: http://localhost:8000/docs
```

### 生产环境部署

```bash
# 1. 配置生产环境变量
cp .env.example .env
# 修改为生产配置

# 2. 构建并启动
make prod-build

# 3. 访问应用
# http://your-domain.com
```

## 📝 使用说明

### 基本流程

1. **注册登录**
   - 访问注册页面创建账号
   - 使用账号登录系统

2. **配置数据源**
   - 添加本地文件、数据库或API数据源
   - 执行同步获取数据

3. **创建知识库**
   - 创建新知识库
   - 上传文档到知识库
   - 系统自动处理和向量化

4. **开始对话**
   - 创建新对话
   - 选择关联的知识库
   - 与AI助手交互

## ⚠️ 注意事项

### 必需配置
1. **OpenAI API Key**: 用于LLM和嵌入向量生成
2. **数据库密码**: 生产环境必须使用强密码
3. **JWT密钥**: 必须修改为随机强密码

### 已知限制
1. 单次上传文件大小限制: 100MB
2. 知识库文档数量建议: < 10,000
3. 并发连接数: 100 (可调整)
